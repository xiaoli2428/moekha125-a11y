# Contributing to moekha125-a11y

Thank you for considering contributing to the moekha125-a11y project! We welcome bug reports, feature requests, documentation improvements, and code contributions.

## How to Contribute

- **Issues:** If you find a bug or have a feature request, please open an issue with clear details and steps to reproduce (if applicable).
- **Pull Requests (PRs):**
  - Fork the repository and create a new branch off of `copilot/link-with-subbase` for your change.
  - Make atomic, well-described commits.
  - Use clear and descriptive PR titles and include a summary of your changes.
  - Reference relevant issues in your PR description if applicable.

## PR Guidelines

- Check that your code builds and passes all tests before opening a PR.
- Ensure your code follows project style and conventions (formatting, naming, documentation, etc).
- Add or update tests as needed for your changes.
- Address existing comments and feedback before requesting re-review.
- Be responsive to comments from maintainers and reviewers.

## Communication

- Be respectful and constructive in discussions.
- Provide context and reasoning for significant changes.
- If you're unsure about anything, feel free to ask before opening a PR!

## Code of Conduct

By participating, you agree to uphold the project's Code of Conduct.

Thank you for helping improve moekha125-a11y!
