# Accessibility Statement

Moekha125-a11y is committed to accessible web experiences. The application supports keyboard navigation, screen readers, semantic HTML, and sufficient color contrast. We periodically review for WCAG 2.1 AA compliance and encourage feedback via GitHub Issues.

## Supported Accessibility Features
- Keyboard navigation throughout the application
- Screen reader compatibility
- Use of semantic HTML elements
- Sufficient color contrast for readability

**Feedback Invitation:**
If you encounter any accessibility barriers or have suggestions for improvement, please open an issue on this GitHub repository. Your feedback helps us make the project more accessible for everyone!
